# Krohnkite Shortcuts

Grow and Shrink Window

```
Meta + L/H
```

Move Window 

```
Meta + Shift + j/k
```

Move Focus Previous / Next

```
Meta + k/j
```

Create and Delete Desktop

```
Alt + -/+
```

Move Window to Desktop (Left - Right)

```
Meta + Shift + Ctrl + Left/Right
```

Shrink or Grow Height
```
Meta + Ctrl + j/k
```
